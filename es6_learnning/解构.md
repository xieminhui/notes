# 变量的解构赋值
从数组和对象中提取值，对变量进行赋值，这被称为解构（Destructuring）。

## 数组的解构
### 基本用法
```
let [a, b, c] = [1, 2, 3];
a;//1
b;//2
c;//3
let [a, b, c] = [1, [2], [3]];
a;//1
b;//[2]
c;//[3]
let [head, ...tail] = [1, 2, 3, 4];
head // 1
tail // [2, 3, 4]
```
不完全解构，左右两边不完全对等
```
let [a, b, c,d] = [1, 2, 3];
d;//undefined;
let [a, [b], c] = [1,[2, 3], 4];
b;//2
```
如果等号的右边不是数组（或者严格地说，不是可遍历的结构,那么将会报错。
```
// 报错
let [foo] = 1;
let [foo] = false;
let [foo] = NaN;
let [foo] = undefined;
let [foo] = null;
let [foo] = {};
```
### 默认值
解构可以给默认值
```
let [a, b, c,d = 4] = [1, 2, 3];
d;//4

let [a, b, c,d = 4] = [1, 2, 3,5];
d;//5
let [x = 1, y = x] = [];     // x=1; y=1
let [x = 1, y = x] = [2];    // x=2; y=2
let [x = y, y = 1] = [];     // ReferenceError: y is not defined,y为什么就使用

```
